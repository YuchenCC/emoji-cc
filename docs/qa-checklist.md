# CC Sticker QA Checklist

Use this checklist before moving a draft into `assets/cc/stickers/`.

## Character Consistency

- Short black hair is preserved.
- Black rectangular glasses are preserved.
- Face remains compact, soft rounded-square / U-shaped, full-cheeked, and friendly.
- Outfit stays close to the light gray sweatshirt / clean casual office-worker look.
- CC does not look like a different character.
- New-theme generation was based only on the approved character and environment references, not previous generated stickers or drafts.

## Sticker Readability

- Emotion or action is understandable without text.
- Face and hands remain readable at small size.
- Pose silhouette is clear.
- The output is one image with one sticker concept, not a collage, grid, multi-panel image, sticker sheet, or set of variants.
- Background matches the sticker theme. If the theme is ambiguous, it uses the standard office workstation context.
- No background clutter competes with the character.

## Image Quality

- Line art is clean.
- Shading is simple and consistent.
- No distorted hands, broken glasses, extra fingers, or melted props.
- No watermark, logo, random letters, or generated text.
- No unrelated extra characters, complex scene clutter, or layout fragments from a collage or sticker sheet.

## Export

- Final asset path follows `assets/cc/stickers/cc-sticker-###-slug.png`.
- `manifest.json` contains the asset record.
- Prompt source is preserved or referenced.
