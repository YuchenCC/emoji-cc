# CC Sticker Prompt Template

Use case: stylized-concept
Asset type: contextual meme sticker
Primary request: Generate one textless meme sticker image of CC expressing `<emotion_or_action>`.

Character:
Use the approved CC design: short black hair, black rectangular glasses, soft rounded-square / U-shaped warm face, clean cute cartoon style, light gray sweatshirt, optional blue lanyard. Keep CC recognizable and consistent.

References:
For each new theme, use only the character standard in `references/cc-character-spec.md`, the approved character reference in `assets/cc/reference/cc-official-reference-v2.png`, the office environment standard in `references/cc-office-environment-spec.md`, and the office environment reference in `assets/cc/reference/cc-office-environment-final.png`. Do not use previous generated stickers in `assets/cc/stickers/` or `assets/cc/drafts/` as image references, style references, composition references, or pose references.

Face contour:
Keep CC's face close to the official source portrait: compact head, broad upper face, full rounded cheeks, short rounded jaw, and small rounded chin. Avoid a long oval face, narrow cheeks, or sharp V-shaped chin.

Pose and expression:
`<pose_expression_detail>`

Composition:
Single character, centered, full-body or upper-body as appropriate, big readable expression, sticker-friendly silhouette, generous padding. Generate exactly one image with one sticker concept only; do not create a collage, grid, multi-panel image, sticker sheet, or multiple variants.

Style:
Clean cute 2D cartoon, smooth bold line art, soft cel shading, bright restrained colors, meme-sticker energy.

Background:
Use a simple contextual background that matches the sticker theme, such as overtime, slacking off, meeting, celebration, collapse, or another relevant office-life scene. If the theme is ambiguous or no stronger scene is specified, use CC's standard bright modern office workstation background from `references/cc-office-environment-spec.md` and `assets/cc/reference/cc-office-environment-final.png`. Keep the background clean, simple, low-clutter, and secondary to CC's expression, pose, gesture, and props.

Text:
No text, no letters, no symbols that look like text.

Avoid:
Photorealism, 3D render, style drift, extra characters, collage layouts, sticker sheets, multi-panel images, complex cluttered backgrounds, logos, watermarks, generated captions.
