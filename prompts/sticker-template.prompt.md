# CC Sticker Prompt Template

Use case: stylized-concept
Asset type: transparent meme sticker
Primary request: Generate one textless meme sticker of CC expressing `<emotion_or_action>`.

Character:
Use the approved CC design: short black hair, black rectangular glasses, soft rounded-square / U-shaped warm face, clean cute cartoon style, light gray sweatshirt, optional blue lanyard. Keep CC recognizable and consistent.

Face contour:
Keep CC's face close to the official source portrait: compact head, broad upper face, full rounded cheeks, short rounded jaw, and small rounded chin. Avoid a long oval face, narrow cheeks, or sharp V-shaped chin.

Pose and expression:
`<pose_expression_detail>`

Composition:
Single character, centered, full-body or upper-body as appropriate, big readable expression, sticker-friendly silhouette, generous padding.

Style:
Clean cute 2D cartoon, smooth bold line art, soft cel shading, bright restrained colors, meme-sticker energy.

Background:
Transparent background preferred. If transparency is unavailable, use a perfectly flat solid chroma-key background for local removal.

Text:
No text, no letters, no symbols that look like text.

Avoid:
Photorealism, 3D render, style drift, extra characters, complex scene backgrounds, logos, watermarks, generated captions.
